# CSS面试相关

  • width的自适应关键字：available、max-content、min-content、fit-content

  • css选择器：id选择器、元素选择器、类选择器、分组（即加逗号）、后代选择器（加空格）、子选择器（>）、相邻（紧随其后的）兄弟选择器（+）、兄弟的所有的某个元素（~、div ~ p）、伪类选择器（p:first-child：表示的是首个p元素，不以p）、伪元素选择器、属性选择器

    优先级：!important > 内联 > id > class / 属性 / 伪类 > 元素选择器 / 伪元素选择器 > 通配选择(*)

  • 继承与不可继承

    • 可继承属性：字体系列、visibility、cursor、text-align、color

  • float：就是让block元素无视float元素，让inline元素让流水一样围绕着float元素来实现浮动布局

    用于文字环绕；行内元素设置float之后可以指定宽高；会高度塌陷；三栏布局

  • BFC，块级格式化上下文(页面上一个隔离的独立容器，容器里面的子元素不会影响到外面的元素)

    规则：

    • 属于同一个BFC的两个相邻box的margin会重叠
    • BFC的区域不会于flot-box重叠
    • 内部浮动，BFC的高度会包括浮动的高度；

    外部浮动，BFC会变窄（自适应），内部浮动，BFC的高度会包括浮动的高度；

    应用：

    • BFC可以用于两栏布局，左侧宽度固定并左浮动，右侧元素开启bfc
    • BFC区域不会与浮动容器发生重叠

    开启：

    • float不为none
    • overflow不为visible
    • position(absolute/fixed)
    • display(inline-block/flex/table-cell)

    解决：1、float高度塌陷 2、外边距重叠

  • line-height：line-height - font-size = 行距

  • link和@import的区别

    @import引入的css在页面加载完毕后被加载；link标签css引入时加载

  • flex属性：默认值是0 1 auto；flex：1是flex-grow 1 flex-shrink 1 flex-basic 0% 的简写形式；flex：auto是1 1 auto

    flex指定数字会被当作 flex:number 1 0;

    flex-basic：定义了在分配多余空间之前，项目占据的主轴空间。默认值是auto，即项目本来的大小；基础值取0%的话，把该弹性元素看成0尺寸，声明宽度是没用的；

  • rem和em的区别

  • display：none（子元素不能显示、不影响css中counter计数、不能点击、不占据）

    visibility：hidden（子元素能显示、影响计数、不能点击、占据）

    opacity：0（子元素不能显示、占据、可以点击、影响计数）的区别

  • 伪元素和伪类的区别：

    伪类：选择那些不能被普通选择器选择的文档之外的元素（表单的伪类：:checked，:disabled）

    伪元素：创建不存在于文档中的元素，只在外部可见

  • css动画硬件加速，是一种性能优化措施，让网页动画变得更好、更流畅

    浏览器中的很多样式都是用cpu计算的，浏览器把内容分到不同图层分别渲染。硬件加速在GPU中进行，更适合做计算量大、每次计算重复量多的工作，单独创建一个层被GPU渲染

    加上will-change：transform/opacity 可以为元素创建新图层，使用GPU加速

    （处理下面的css的时候，会使用GPU渲染，3D、transform、filter、opacity、）

  • rem适配的原理：rem单位表示屏幕宽度跟html元素的fontsize成正比；rem单位和px单位进行换算

  • requestAnimation，请求动画帧，要求浏览器在下次重绘之前调用指定的回调函数更新动画。该方法需要传入一个回调函数作为参数，该回调函数会在浏览器下一次重绘之前执行。 该方法属于宏任务。

    优势：CPU节能，当页面处于未激活状态下，该页面的屏幕刷新任务也会被系统暂停，跟着系统走的requestAnimation也会停止渲染。函数节流，回调函数控制在一段时间内执行一次。会将每一帧的所有dom操作都集中起来，减少重绘和重排

  • sticky：粘性定位，在目标区域内像position:relative，某个元素距离其父元素距离达到要求时，就像是position:fixed

    使用条件：父元素不能overflow：hidden；必须指定top、bottom、left、right四个值之一；父元素高度不能低于sticky元素的高度

    使用注意：相对最近的滚动元素定位，contain的元素高度不够，sticky元素会被拖走，相对的滚动元素和产生滚动的元素是一致

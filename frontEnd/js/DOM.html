# 常用DOM操作

# Node类型

在JavaScript中，所有节点类型都继承Node类型。

每个节点都有nodeType、nodeName（标签名）、childNodes（NodeList类数组）、还有一些能表示节点关系的属性（nextSiBling、perviousSiBling、firstChild、lastChild）

操纵节点的方法：appendChild（末尾添加新节点，返回新节点）、insertBefore(要插入的节点， 参照节点)、removeChild（传入要移除的节点）、cloneNode（返回与调用它的节点一摸一样的节点，接收一个布尔值表示是否进行深复制）

# Document类型

表示文档节点的类型。浏览器文档对象document是HTMLDocument的实例，表示整个HTML页面，也是window对象的属性

常用属性：document.documentElement（拿到html元素），docume.body（拿到body元素）

常用定位元素的方法：ElementById（id属性）、ElementsByTagName（标签名，返回一个类数组）、ElementsByName（name属性）

# Element类型

表示HTML元素，对外暴露访问元素的标签名、子节点、属性的能力

HTML元素上常用属性：id、title、className

取得HTML元素属性的方法：getAttribute、setAttribute、removeAttribute

创建元素：document.createElement（传入创建元素的标签名）

# Text类型

表示纯文本节点

创建文本节点：document.createTextNode（传入要插入的文本）

# DOM扩展

# Selectors API

暴露在document类型和element类型的实例身上

1、querySelector()，接收css选择符参数，返回匹配该模式的第一个后代元素

2、querySelectorAll()，接收查询css的参数，返回所有匹配的节点，一个NodeList的静态实例

# Element Traversal API

1、childrenElementCount：返回子元素数量，不包括文本节点和注释

2、firstElementChild：指向第一个Element类型的子元素，不会受文本节点的影响，下面的属性同理

3、lastElementChild

4、previousElementSiBling

5、nextElementSiBling

# HTML5扩展

1、getElementsByClassName，暴露在document对象和所有HTML元素身上

2、classList属性，通过className属性实现添加、删除和替换。传入一个类名，用add、remove、toggle、contains方法，用于增删查等操作

3、焦点管理。按钮、输入框的focus方法，hasfocus方法检查是否拥有焦点

4、HTMLDocument扩展：readyState表示这个文档的加载状态

5、插入标记。innerHTML，浏览器会解析设置的html字符串，所以可能设置完再读innerHTML，结果会不一样。

6、scrollIntoView，存在于所有HTML元素上，用于确保元素可见

# 专有扩展

1、children属性，只包含Element类型的子节点

2、contains方法，确定一个元素是不是另一个元素的后代

3、innerText：会移除所有后代节点，完全改变DOM子树，会编码出现再字符串中出现的HTML语法字符（小于号、大于号、引号以及和号）。innerText可以用于去除HTML标签，div.innerText = div.innerText; 去除HTML标签

# 偏移尺寸

offetHeight：元素在垂直方向上占用的像素尺寸，包括它的高度、水平滚动条高度和上、下边框的高度

offsetTop：元素上边框外侧距离包含元素上边框内侧的像素数

offsetTop、offsetWidth是相对于包含元素的，包含元素保存在offsetParent属性中

function getElementLeft(element){
    let actualLeft = element.offsetLeft;
    let current = element.offsetParent;
    while(current !== null){
        actualLeft += current.offsetLeft;
        current = current.offserParent;
    }
    return actualLeft;
}

# DOM事件

# HTML事件处理程序

特定元素支持的每个事件都可以使用事件处理程序的名字以HTML属性的形式命名。

属性的值是JS代码，不能在未经转义的情况下使用HTML语法字符

也可以指定一个定义过的方法，方法可以在script标签定义，也可以来自外部js文件

# DOM0 事件处理程序

获取元素，给元素的事件处理程序属性赋予一个方法。

# DOM2 事件处理程序

通过addEventListener和removeEventListener，接收3个参数：事件名、事件处理函数和一个布尔值（true表示捕获阶段、false表示冒泡阶段），可以为同一事件处理程序添加多个函数

# IE事件处理程序

IE实现了两个方法：attachEvent、detachEvent，接收事件处理程序的名字和事件处理函数，只支持添加到冒泡阶段

和DOM事件相同，可以为同一事件同时添加多个函数，但会以添加他们的顺序反向触发

# 事件对象

在DOM中发生事件时，所有相关信息会被收集并存储在一个名为event的对象中。

# DOM事件对象

event.type、event.target（真正点击的元素）、event.currentTarget（设置事件处理程序的节点）、event.preventDefault()、event.stopPropagation()

# IE事件对象

event.cancelBubble、event.returnValue（默认为true，设置为false可以取消事件默认行为）、event.srcElement(与DOM的target相同)
